# Global Crisis Index Data

This is the official dataset from [GCIS Fusion](https://globalcrisisindex.com).

# Contents
- 60+ countries with real-time crisis urgency scores (0-100)
- 8-dimension scoring: conflict, displacement, food security, health, economic, climate, access, political
- Crisis types (earthquake, flood, conflict, drought, epidemic, etc.)
- Critical needs (food, shelter, health, water, protection)

# Data Sources
Aggregated from live APIs including USGS, NASA, ACLED, WHO, FAO, World Bank, and UNHCR.
