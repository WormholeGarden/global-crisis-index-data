# Global Crisis Index (GCIS) — Live Crisis Data

## Description
Real-time crisis intelligence data for 60+ countries, updated every 5 minutes from 40+ live APIs.

## Data Fields
- `country_code`: ISO 3-letter country code
- `country_name`: Full country name
- `urgency_score`: 0-100 composite score
- `severity`: Critical (80+), High (60-79), Moderate (40-59), Low (0-39)
- `active_crises`: Array of crisis types (earthquake, conflict, flood, drought, epidemic, etc.)
- `needs`: Primary humanitarian needs (food, shelter, health, water, protection)
- `timestamp`: Last update time (UTC)

## Data Sources
USGS, NASA EONET, ACLED, GDACS, ReliefWeb, WHO, FAO, FEWS NET, IPC, WFP, World Bank, IMF, UNHCR, NOAA, Open-Meteo, GDELT
